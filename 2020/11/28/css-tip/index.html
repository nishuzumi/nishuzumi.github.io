由于position:absolute是根据上一个非static定位的元素。所以需要设定.tip元素为position:relative，这个设定并没有固定的要求，可以在hover里面设定，也可以直接设定在.tip上。

设置内容为属性内容，定位为绝对定位，并且设定和最下方之间的距离。

这个部分的内容是用于绘制三角形的内容，通过绘制一个正方形，然后不绘制左边和上面的边，再旋转元素，并且使用一样的位移手段。
